# Week 2 Summary

## Overview

Week 2 of the Thunder Cipher Cybersecurity Internship focused on networking fundamentals, OSINT, active reconnaissance, scanning, enumeration, and introductory vulnerability assessment. The week combined theory with hands-on labs, helping bridge the gap between basic Linux knowledge and practical cybersecurity workflows.

---

## Sessions Covered

### Day 4 - Networking Fundamentals and Reconnaissance
- OSI model and its security relevance
- TCP vs UDP
- TCP three-way handshake
- ARP spoofing
- Common ports and protocols
- Passive reconnaissance
- Nmap basics
- DNS tools such as `dig`, `nslookup`, `arp`, and `ip neigh`
- Packet capture using `tcpdump` and Wireshark

### Day 5 - DNS, DHCP, NAT, and Subnetting
- DNS resolution process
- DNS record types
- DHCP and the DORA process
- NAT and its types
- CIDR notation and subnetting
- DNS enumeration
- Packet analysis using Wireshark and `tcpdump`

### Day 6 - OSINT and Reconnaissance
- Passive vs active reconnaissance
- WHOIS
- DNS reconnaissance
- Shodan
- theHarvester
- Building a target profile from public sources
- ThunderCipher OSINT challenge

### Day 7 - Scanning and Enumeration
- Scanning vs enumeration
- Nmap scan types
- Service and version detection
- OS fingerprinting
- Banner grabbing
- Nmap Scripting Engine (NSE)
- Gobuster directory enumeration
- ThunderCipher labs

### Day 8 - Vulnerability Scanning and Web Exploitation Basics
- Nessus installation and basic network scan
- Vulnerability assessment workflow
- Web2Shell lab
- HTTP 403 bypass concepts
- WAF rules
- Reverse shell fundamentals
