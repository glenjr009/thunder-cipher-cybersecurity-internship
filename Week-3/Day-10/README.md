## Theory Notes

### Burp Suite Professional

Burp Suite Professional is one of the most widely used web application security testing tools. It acts as an intercepting proxy, allowing security professionals to inspect, modify, and analyze HTTP and HTTPS traffic between a browser and a web application.

---

### FoxyProxy

FoxyProxy is a browser extension used to manage proxy configurations. During today's session, it was configured to route browser traffic through Burp Suite using the local proxy address:

- **Host:** `127.0.0.1`
- **Port:** `8080`

This allows Burp Suite to intercept and inspect web traffic generated by the browser.

---

### Burp Suite Professional vs Community Edition

We discussed the key differences between the Community and Professional editions of Burp Suite.

| Community Edition | Professional Edition |
|-------------------|----------------------|
| Basic manual testing | Advanced automation features |
| Limited Intruder speed | High-speed Intruder |
| No Scanner | Built-in Vulnerability Scanner |
| Basic functionality | Advanced extensions & automation |
| Suitable for beginners | Preferred by security professionals |

---

### Burp Suite Modules

The following Burp Suite modules were introduced:

- Proxy
- Target
- Repeater
- Intruder
- Decoder
- Comparer
- Logger

Each module serves a different purpose during web application security assessments.
