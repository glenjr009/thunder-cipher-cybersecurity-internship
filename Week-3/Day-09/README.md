# Day 09 - Web Security Fundamentals & JWT Authentication

**Date:** 15 July 2026

---

## Objective

The objective of today's session was to understand the fundamentals of web security, common web authentication mechanisms, session management, and JSON Web Tokens (JWT). The session concluded with hands-on practice using Burp Suite and PortSwigger Web Security Academy labs.

---

## Topics Covered

- CVE (Common Vulnerabilities and Exposures)
- CVSS (Common Vulnerability Scoring System)
- CVSS Severity Ratings
- Web Fundamentals
- HTTP vs HTTPS
- Cookies
- Session Management
- Session Hijacking (Concept & Demonstration)
- Cookie Editor Extension
- JSON Web Tokens (JWT)
- JWT Structure and Signature
- Burp Suite
- JWT Editor Extension
- PortSwigger Web Security Academy

---

## Practical Activities

- Learned how vulnerabilities are identified and classified using CVE and CVSS.
- Understood how cookies are used to maintain authenticated sessions.
- Observed a demonstration explaining session hijacking concepts in an authorized learning environment.
- Explored how browser extensions such as Cookie Editor can be used to inspect and modify cookies for educational purposes.
- Learned the structure of JWTs, including the Header, Payload, and Signature.
- Installed the JWT Editor extension in Burp Suite.
- Used Burp Suite Repeater to modify JWTs during an authorized lab exercise.
- Completed a JWT Authentication lab on the PortSwigger Web Security Academy platform.

## Key Learnings

- Understood the purpose of CVE and CVSS in vulnerability management.
- Learned how web applications use cookies for authentication.
- Developed a conceptual understanding of session hijacking and why secure session management is important.
- Explored how JWT-based authentication works.
- Learned the structure and role of JWT headers, payloads, and signatures.
- Practiced intercepting and modifying requests using Burp Suite in an authorized lab.
- Successfully completed a JWT authentication lab on PortSwigger.

## Reflection

Today's session provided valuable insight into web application security and modern authentication mechanisms. Understanding how sessions, cookies, and JWTs work helped explain why secure authentication and proper session management are critical for protecting web applications. The hands-on exercises using Burp Suite and PortSwigger reinforced these concepts by demonstrating how authentication flows can be analyzed and tested in a safe, educational environment.
