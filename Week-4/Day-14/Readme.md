# Reconnaissance

Reconnaissance is the process of collecting publicly available information about a target before performing any testing.

Examples include:

- Discovering IP addresses
- Identifying running services
- Gathering DNS information
- Collecting website information

Reconnaissance forms the foundation of every penetration test.

---

# Enumeration

Enumeration involves actively interacting with the target to gather detailed information.

Examples include:

- Open ports
- Running services
- User accounts
- Shared resources
- Web directories

Accurate enumeration often determines the success of a penetration test.

---

# Information Gathering

Information gathering combines passive and active techniques to understand the target environment.

Examples:

- Domain information
- Network services
- Operating system details
- Web technologies
- Usernames
- Service versions
